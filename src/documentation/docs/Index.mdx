This <a class="ui-text-espirit" href="https://vuejs.org/" target="_blank">Vue.js</a> project provides basic components and complete sections.
These can be easily installed via npm and included in your Vue project.

These components help you to set up your own project quickly.

The project is part of the **FirstSpirit Experience Accelerator (FSXA)**.

<br />

### About the FSXA
The FirstSpirit Experience Accelerator (FSXA) is the hybrid solution of a digital 
experience platform, combining a headless approach with enterprise capabilities. 

If you are interested in the FSXA check this <a class="ui-text-espirit" href="https://docs.e-spirit.com/module/fsxa/overview/benefits-hybrid/" target="_blank">Overview</a>.
You can <a class="ui-text-espirit" href="https://www.e-spirit.com/us/specialpages/forms/on-demand-demo/" target="_blank">request a demo online</a>.
